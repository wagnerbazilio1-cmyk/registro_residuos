🌐 Como publicar para colegas
✅ Pré-requisito: Apps Script publicado corretamente
  1. No Google Sheets → Extensões → Apps Script
  2. Cole o código (botão 📋 Copiar código no sistema)
  3. Implantar → Nova implantação → Aplicativo da Web
  4. Executar como: Eu
  5. Quem tem acesso: Qualquer pessoa (sem "com Google")
  6. Copie a URL gerada e cole em ⚙ Google Sheets no sistema
Escolha como publicar o HTML:
🐙
GitHub Pages GRATUITO · RECOMENDADO
Link: https://wagnerbazilio1-cmyk.github.io/Gestao-residuos
  1. Acesse github.com e crie uma conta gratuita
  2. Clique em New repository → nome: Gestao-residuos
  3. Marque Public e clique em Create repository
  4. Clique em uploading an existing file
  5. Arraste o arquivo residuos_v6.html e renomeie para index.html
  6. Clique em Commit changes
  7. Vá em Settings → Pages → Branch: main → Save
  8. Aguarde 1-2 min → seu link estará disponível!
🌐
Google Sites GRATUITO · USA CONTA GOOGLE
Link: https://sites.google.com/view/sistema-residuos
  1. Acesse sites.google.com e clique em + (novo site)
  2. Dê um nome ao site (ex: Sistema de Resíduos)
  3. Clique em Inserir → Incorporar
  4. Escolha Incorporar código e cole o conteúdo do HTML dentro de uma tag iframe
  5. ⚠️ Atenção: Google Sites tem limitações com HTML complexo. Prefira GitHub Pages.
⚠️ Google Sites não permite hospedar HTML diretamente. Use o Google Drive como CDN: faça upload do HTML no Drive, compartilhe publicamente e use o link direto — mas alguns navegadores bloqueiam isso. GitHub Pages é mais confiável.
